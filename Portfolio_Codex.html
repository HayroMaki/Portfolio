Projet Codex Naturalis

Retour

Codex Naturalis est un jeu de société dans lequel on doit placer des cartes dans les coins d'autres cartes, obtenir des ressources qui permettent de placer des cartes dorées, ayant un coût mais donnant des points au joueur, obtenir 20 points offre la victoire.

Dans le cadre de ce projet, moi et mon binôme avons donc reproduit ce jeu en Java via la bibliothèque fournie pour le projet mais nous n'avons pas implémenté le côté multijoueur du jeu originel.

Vous pouvez télécharger et essayer le jeu ici.
